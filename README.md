- 사용자가 겪은 사용자가 겪은 소비자 피해 상황을 입력하면 피해 유형, 법적 근거, 대응 방법을 자동으로 분석해주는 서비스
- 엑셀 파일(.csv) 업로드 기능 및 텍스트 입력 기능 구현
- 디자인: 공정거래위원회의 정체성을 담은 파란색 바탕
- 결과서는 한글파일(hwp, hwpx) 또는 PDF 파일 다운로드 가능해야된다.
- 출력 항목: 
피해 유형 (예: 환불 거부)
카테고리 (전자상거래/서비스 등)
위험도 (낮음/중간/높음)등이 포함 되어야 하며, 피해 상황에 대한 요약 기능도 제공.
- 메인 화면에는 해당 내용에 대한 도넛 차트로 대시보드 구현
- 외부 URL 버튼으로 신고 기능 추가 및 신고 문장 자동 생성 기능 추가
